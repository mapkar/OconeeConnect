OconeeConnect Connection Instructions

Join the local mesh network in the Oconee/Charlotte area—easy setup with default settings!

Quick Overview

OconeeConnect uses Meshtastic's default settings for simplicity: the primary channel is "LongFast" with default encryption. This means your device can connect out-of-the-box without custom configs. Just get in range of other nodes, and you're part of the mesh!

Default Settings We're Using:
  • Channel: Primary (Channel 0), named "LongFast".
  • Preset: Long Fast (good balance of range and speed—up to several miles line-of-sight).
  • Encryption: Default PSK ("AQ==")—it's basic encryption with a public key, so anyone on defaults can join. (For private chats, use secondary channels or DMs.)
  • Region: US (915 MHz band)—auto-detected based on your location.

If you're new, expect to send/receive text messages and share locations off-grid. Great for local events, emergencies, or just testing in Charlotte!

Step-by-Step Setup

  1. Get Hardware: Buy a Meshtastic-compatible LoRa device (e.g., Heltec V3, LilyGO T-Beam, RAK WisBlock—$20–$50 on Amazon, AliExpress, or stores like Rokland). Many come pre-flashed with Meshtastic.
  2. Install Firmware (if needed):
    • Go to flasher.meshtastic.org in a web browser (Chrome/Edge).
    • Connect your device via USB.
